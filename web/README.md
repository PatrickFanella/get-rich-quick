# Get Rich Quick Web

React 19 + TypeScript + Vite frontend scaffold for the Get Rich Quick trading agent.

## Scripts

- `npm run dev` – start the development server
- `npm run build` – build the production bundle
- `npm run lint` – run ESLint
- `npm test -- --run --reporter=verbose` – run the Vitest suite

## Environment

Set `VITE_API_BASE_URL` to point at the backend API. It defaults to `http://localhost:8080`.
